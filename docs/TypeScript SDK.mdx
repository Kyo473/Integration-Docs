<Admonition type="tip" icon={<FaCheckCircle />} title="Official SDK">
 SDK является оффицальной библиотекой от команды infomaximum
</Admonition>

Позволяет типизированно писать интеграции

Реализован класс URL для парсинга и манипуляции адресами внутри интеграций.
Builder для создания API клиента, позволяет гибко настраивать HTTP,GraphQL и кастомные клиенты.
Перехватчик ошибок onHttpError и onNetworkError

import Admonition from '@theme/Admonition';
import { FaCheckCircle } from "react-icons/fa";

## Установка

```bash
npm install @infomaximum/integration-sdk
```
или 
```bash
yarn add -D @infomaximum/integration-sdk
```
:::warning
Всегда выбирайте и закрепляйте правильную версию SDK, соответствующую версии proceset
:::
